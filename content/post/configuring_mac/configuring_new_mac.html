Configuring a new mac

  1. Download Chrome
  2. Download Sublime
  3. Install Homebrew
  4. brew install python
  5. brew install R
  6. brew cask install Rstudio
  7. brew cask install pycharm
  8. brew cask install java
  9. brew cask install dbeaver-community
  10. brew cask install slack
  11. brew cask install spectacle
  12. Configure VPN
  13. brew install git
  14. Create ssh key: ssh-keygen -t rsa -b 4096 (https://docs.typo3.org/m/typo3/guide-contributionworkflow/master/en-us/Appendix/OSX/SSHKeyOSX.html)
  15. Save key in GitHub
  16. Clone repositories
  17. brew cask install clipy
  18. brew cask install docker
